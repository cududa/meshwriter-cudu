MeshWriter Examples

These examples demonstrate MeshWriter v2.0.0 with Babylon.js 8+ support.

Before Running Examples

You need to build the MeshWriter distribution first:

cd /path/to/meshwriter
npm install
npm run build

This creates dist/meshwriter.min.js which the examples load.

Examples

  • Basic Example

    Simple "Hello World" text - the minimal setup to get started.

  • Holes Test (CSG2 Verification)

    Tests letters with holes: B, O, 8, %, @. This verifies that CSG2 boolean operations work correctly. If letters appear garbled, CSG is broken.

  • Styled Text

    Multiple text styles with custom colors, sizes, and material properties.

  • Animated SPS

    Each letter rotates independently using SolidParticleSystem animation.

What to Look For

When testing, check for:

  • Letters with holes (B, O, 8, %, @) - holes should be punched out correctly
  • No garbled/broken text - all letters should render cleanly
  • Performance - text should render quickly without noticeable delay
  • Console output - check for CSG version info and any errors

Babylon.js 8+ API

These examples use the new async API:

// For Babylon 8+
const Writer = await BABYLON.MeshWriter.createAsync(scene, { scale: 1 });

// Helper methods
BABYLON.MeshWriter.getCSGVersion()  // Returns 'CSG2', 'CSG', or null
BABYLON.MeshWriter.isReady()        // Returns true if CSG is initialized

Custom CSG Initialization

If your build tree-shakes Babylon globals, register an initializer and readiness check before calling createAsync:

import { InitializeCSG2Async, IsCSG2Ready } from "@babylonjs/core/Meshes/csg2";

BABYLON.MeshWriter.setCSGInitializer(() => InitializeCSG2Async(scene));
BABYLON.MeshWriter.setCSGReadyCheck(() => IsCSG2Ready());

const Writer = await BABYLON.MeshWriter.createAsync(scene, { methods: methodsObj });
